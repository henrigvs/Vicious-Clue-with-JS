Rules

____________________________________

During this game, you will have to answer to some riddles.
You need to provide the correct answer to access to the next riddle.
If you can't find the correct answer
you can request a clue that could certainly help you.

To be able to play, you've to authenticate you in the login page
or create an account if you're not registered yet.

Once logged in, you're going to be able to play, create enigmas and manage it.
You can also challenge others players and see your ranking on ranking page.

Formula to set your score :
Incorrect answer : - 20 points
Clue requested : - 10 points
Correct answer : + 50 points
Series of 3 : + 100 points
